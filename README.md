# Smart Study Generator Agent

## Description

An AI-powered Smart Study Generator Agent built using LangFlow and Google Gemini 3.5 Flash.

## Features

- Personalized Study Plans
- Concept Summaries
- Quiz Generation
- Flashcards
- Study Scheduling

## Technologies Used

- LangFlow
- Google AI Studio
- Google Gemini 3.5 Flash
